# App Store Listing Draft

## App Name

GradeFlip

## Subtitle

Flashcards, study buddies, and AI help

## Promotional Description

Build decks, study offline, sync when you want, and use AI help only when you choose.

## Description

GradeFlip is a native Apple study app for iPhone, iPad, and Mac.

Use the paid core app to build flashcard decks, attach images, keep per-card notes, and study offline with layouts designed for each Apple platform.

Upgrade with the optional online layer for cloud sync, study buddies, deck sharing, messaging, likes, and friend streaks.

Use AI help separately with purchasable study tokens. Choose deck-grounded mode for responses constrained to your selected decks, or use the metered web-enabled mode when broader help is needed.

## Monetization Copy Rules

- Paid app purchase unlocks the offline core.
- Monthly subscription unlocks online sync and social features.
- Study tokens are a separate consumable for AI usage.
- Expired subscriptions do not remove owned offline study content.
