# Privacy Disclosures

## Stored Data

- Local decks, cards, notes, and image attachments are stored on device.
- Online mode can store accounts, synced deck metadata/content, social events, and subscription-linked activity.
- AI usage can store usage metadata, token ledger records, and provider/model attribution.

## User-Facing Privacy Copy

- Offline study content remains available without creating an account.
- Cloud sync requires an account and stores synced study data on the selected backend.
- Social features store buddy relationships, shares, requests, likes, and related event history.
- AI features meter provider usage and token balances but should avoid storing unnecessary prompt content.

## Permission Copy

- Note dictation: used only to turn speech into note text at the user’s request.
- Cloud sync: used to keep your decks available across signed-in devices.
- Messaging/social: used to exchange deck shares, requests, likes, and buddy activity.
- AI usage: used to process AI study requests and meter study-token consumption.
