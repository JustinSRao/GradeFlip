# Epic 05: AI tutor and token economy

## Overview

Deliver GradeFlip's AI study assistant as a multi-provider system that supports deck-grounded chat, optional web-enabled answers, and a paid token economy that is understandable to users and financially safe for the product.

## Success Criteria

- [ ] GradeFlip can route AI requests through a shared abstraction across supported providers
- [ ] Users can choose deck-grounded chat or web-enabled chat modes
- [ ] AI usage is charged with predictable internal study-token pricing
- [ ] Token gifting and usage history are supported for online accounts
- [ ] The token-pricing system estimates cost without requiring an extra AI call

| Sprint | Title | Status |
|--------|-------|--------|
| 13 | Multi-provider AI abstraction with deck-grounded chat | planned |
| 14 | Web-enabled AI chat, study tokens, gifting, and usage metering | planned |

## Backlog

- [ ] Define supported provider capabilities and fallback behavior
- [ ] Define cost-estimation and reconciliation rules for study tokens
- [ ] Define token gift flows and transaction history views
