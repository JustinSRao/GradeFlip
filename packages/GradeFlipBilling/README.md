# GradeFlipBilling

This package will host monetization and capability-gating logic for GradeFlip.

Planned responsibilities:

- paid app entitlement handling
- online subscription entitlement handling
- AI token product metadata
- feature capability evaluation
- StoreKit-facing business rules shared across the app

Current Sprint 3 deliverables:

- app-mode definitions for locked, offline, upsell, and subscriber states
- placeholder product catalog entries for paid core, monthly subscription, and AI token packs
- capability lock reasons and purchase-surface recommendations
- app-facing billing scaffold inputs for later StoreKit 2 integration
